HOW TO RUN MY PROGRAM
 1. download and install Anaconda
 2. unzip program folder and save ro desired directory.
 3. Launch "anaconda prompt IDE" from All-programs in your system
 4. On the interface, type Jupyter Notebook, and click enter.
 5. on the browser that opens, navigate to the folder where you stored the unzipped program.
 6. click on the model.ipynb file and open it on the browser.
 7. Run the program from first block, one block at a time.
	to know if a block has finished running, the asterix in the squared bracket will be replaced with a number,
	then run the next block, until the last block
NOTE: I have included descriptions to describe what the codes are doing for each section, sothat you can 
understand the process.
